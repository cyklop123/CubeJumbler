# CubeJumbler

Speedcuberze!
Nie musisz martwić się o to czy dobrze pomieszałeś kostkę rubkia. CubeJumbler zrobi to za ciebie.

1. Wpisz scramble
2. Kliknij przycisk "Scramble!"
3. Sprawdź wygenerowany szablon z pomieszaną kostką.
